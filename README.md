\# 💈 Barbearia PDV



Sistema web de ponto de venda (PDV) para barbearia, desenvolvido com Django, com foco na organização de atendimentos, clientes e serviços.



\---



\## 🚀 Funcionalidades



\* 👤 Cadastro de clientes

\* ✂️ Cadastro de serviços

\* 📅 Agendamento de atendimentos

\* 🔄 Controle de status dos pedidos (atendimento, corte, finalizado, etc.)

\* 🧾 Organização do fluxo de produção



\---



\## 🛠️ Tecnologias utilizadas



\* Python

\* Django

\* HTML / CSS

\* SQLite



\---



\## 📂 Estrutura do projeto



barbearia-pdv/

│

├── barbearia\_pdv/ → configurações do projeto

├── pdv/ → app principal

├── manage.py → gerenciador do Django

├── requirements.txt → dependências

├── README.md → documentação



\---



\## ⚙️ Como rodar o projeto



```bash

\# Clonar o repositório

git clone https://github.com/matheusgabrieldev/barbearia-pdv.git



\# Entrar na pasta

cd barbearia-pdv



\# Criar ambiente virtual

python -m venv venv



\# Ativar ambiente (Windows)

venv\\Scripts\\activate



\# Instalar dependências

pip install -r requirements.txt



\# Rodar migrações

python manage.py migrate



\# Iniciar servidor

python manage.py runserver

```
